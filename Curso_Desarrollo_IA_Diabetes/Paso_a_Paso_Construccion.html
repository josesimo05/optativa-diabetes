Guía de Construcción: Diamond Diabetes AI

1
Crear la Base del Proyecto

Lo primero es crear una carpeta limpia donde vivirá todo nuestro código.

# Abre tu terminal y ejecuta uno por uno:
mkdir diabetes-ai
cd diabetes-ai
mkdir backend
✅ Verifica que estés dentro de la carpeta "diabetes-ai".
2
Preparar el Entorno Python (Backend)

Vamos a crear un "Entorno Virtual" para instalar las librerías de IA sin afectar tu sistema principal.

cd backend
python -m venv venv
# En Windows:
venv\Scripts\activate
# En Mac/Linux: source venv/bin/activate
✅ Verifica que veas (venv) en color verde al inicio de tu terminal.
3
Instalar Librerías de Inteligencia Artificial

Crea un archivo llamado requirements.txt dentro de la carpeta backend y pega la lista de ingredientes:

fastapi uvicorn scikit-learn pandas numpy pydantic

Ahora, dile a Python que instale todo:

pip install -r requirements.txt
4
Escribir el Código del Cerebro (API)

Crea un archivo llamado main.py dentro de backend. Este es el corazón del proyecto.

from fastapi import FastAPI from pydantic import BaseModel from sklearn.datasets import load_diabetes from sklearn.linear_model import LinearRegression import uvicorn app = FastAPI() # 1. ENTRENAR EL MODELO AL INICIAR print("Entrenando IA...") diabetes = load_diabetes(scaled=False) X, y = diabetes.data, diabetes.target model = LinearRegression() model.fit(X, y) # 2. DEFINIR DATOS DE ENTRADA class Paciente(BaseModel): age: float sex: float bmi: float bp: float s1: float s2: float s3: float s4: float s5: float s6: float # 3. CREAR RUTA DE PREDICCIÓN @app.post("/predict") def predecir(data: Paciente): datos = [[ data.age, data.sex, data.bmi, data.bp, data.s1, data.s2, data.s3, data.s4, data.s5, data.s6 ]] resultado = model.predict(datos) return {"progreso_estimado": resultado[0]} # 4. ARRANCAR EL SERVIDOR if __name__ == "__main__": uvicorn.run(app, host="0.0.0.0", port=8000)

Pruébalo ahora mismo:

python main.py
✅ Deberías ver: "Uvicorn running on http://0.0.0.0:8000"
5
Crear la Interfaz Web (Frontend)

Deja corriendo el backend. Abre una NUEVA TERMINAL y vuelve a la carpeta raíz.

# Asegúrate de estar en /diabetes-ai
npx create-vite@latest frontend --template react-ts
cd frontend
npm install
6
Conectar Todo (App.tsx)

Abre el archivo frontend/src/App.tsx. Borra todo lo que hay dentro y pega este código final:

import { useState } from 'react'; function App() { const [form, setForm] = useState({ age: 50, sex: 1, bmi: 25, bp: 80, s1: 150, s2: 100, s3: 50, s4: 4, s5: 4, s6: 90 }); const [resultado, setResultado] = useState(null); const enviar = async () => { const res = await fetch('http://localhost:8000/predict', { method: 'POST', headers: {'Content-Type': 'application/json'}, body: JSON.stringify(form) }); const data = await res.json(); setResultado(data.progreso_estimado); }; return ( <div style={{padding: '50px', background: '#f8fafc', minHeight: '100vh'}}> <h1>Diabetes AI</h1> <div style={{display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '15px'}}> {Object.keys(form).map(campo => ( <div key={campo}> <label style={{display:'block', fontWeight:'bold'}}>{campo.toUpperCase()}</label> <input type="number" value={form[campo]} onChange={e => setForm({...form, [campo]: parseFloat(e.target.value)})} style={{padding: '10px', width: '100%'}} /> </div> ))} </div> <button onClick={enviar} style={{marginTop: '30px', padding: '15px', background: '#2563eb', color: 'white', border: 'none', cursor: 'pointer', width:'100%'}}> CALCULAR RIESGO AHORA </button> {resultado && ( <div style={{marginTop: '30px', padding: '20px', background: '#dcfce7', color: '#166534', border: '1px solid #166534'}}> <h2>Resultado: {resultado.toFixed(2)}</h2> <p>Progresión estimada de la enfermedad a 1 año.</p> </div> )} </div> ); } export default App;

¡Momento de la verdad! Ejecuta el frontend:

npm run dev
