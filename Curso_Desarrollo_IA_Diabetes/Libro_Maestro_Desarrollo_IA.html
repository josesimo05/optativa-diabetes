4

Programación del Algoritmo (IA)

Ahora escribiremos el código que aprende. Crea el archivo main.py en la carpeta backend.

Acción del Alumno: Copiar y Pegar Código Maestro

/backend/main.py
from fastapi import FastAPI
from pydantic import BaseModel
from sklearn.datasets import load_diabetes
from sklearn.linear_model import LinearRegression
import uvicorn

app = FastAPI()

# 1. CARGA DE DATOS & ENTRENAMIENTO
print("Cargando cerebro...")
diabetes = load_diabetes(scaled=False)
X, y = diabetes.data, diabetes.target
model = LinearRegression()
model.fit(X, y)
print("Modelo entrenado exitosamente.")

# 2. DEFINICION DEL CONTRATO DE DATOS
class Paciente(BaseModel):
    age: float
    sex: float
    bmi: float
    bp: float
    s1: float
    s2: float
    s3: float
    s4: float
    s5: float
    s6: float

# 3. ENDPOINT DE PREDICCION
@app.post("/predict")
def predict(data: Paciente):
    datos_paciente = [[
        data.age, data.sex, data.bmi, data.bp,
        data.s1, data.s2, data.s3, data.s4, data.s5, data.s6
    ]]
    prediccion = model.predict(datos_paciente)
    return {"resultado": prediccion[0]}

# 4. LANZADOR
if __name__ == "__main__":
    uvicorn.run(app, host="0.0.0.0", port=8000)

Prueba Rápida: Ejecuta python main.py. Si ves "Modelo entrenado exitosamente", ¡funciona!

Acción del Alumno:

Abre el archivo frontend/src/App.tsx. Borra TODO su contenido y pega esto:

import { useState } from 'react';
import './App.css';

function App() {
  // ESTADO: Aquí guardamos los datos del formulario
  const [form, setForm] = useState({
    age: 50, sex: 1, bmi: 25, bp: 80, 
    s1: 150, s2: 100, s3: 50, s4: 4, s5: 4, s6: 90
  });
  const [resultado, setResultado] = useState(null);

  // FUNCIÓN: Escucha cambios en los inputs
  const handleChange = (e) => {
    setForm({...form, [e.target.name]: parseFloat(e.target.value)});
  };

  // FUNCIÓN: Envía los datos a Python
  const predecir = async () => {
    const respuesta = await fetch('http://localhost:8000/predict', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(form)
    });
    const data = await respuesta.json();
    setResultado(data.resultado);
  };

  // VISUAL (JSX)
  return (
    <div className="card">
      <h1>Diabetes AI</h1>
      <div style={{display: 'grid', gridTemplateColumns: '1fr 1fr', gap: '10px'}}>
        {Object.keys(form).map(key => (
            <div key={key}>
                <label>{key.toUpperCase()}</label>
                <input name={key} type="number" value={form[key]} onChange={handleChange} />
            </div>
        ))}
      </div>
      <button onClick={predecir} style={{marginTop: '20px'}}>CALCULAR PROGRESIÓN</button>
      
      {resultado && (
          <h2>Índice de Riesgo: {resultado.toFixed(2)}</h2>
      )}
    </div>
  );
}

export default App;

Paso Final: Ejecuta npm run dev en la terminal del frontend. ¡Listo!
